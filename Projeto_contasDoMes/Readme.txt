Contas do Mês
Este projeto consiste em um simples programa web para o registro e exibição de contas do mês. Os usuários podem inserir a descrição e o valor de uma conta, e o programa calculará o total e exibirá a lista de contas registradas.

Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

HTML: Linguagem de marcação utilizada para estruturar a página web.
CSS: Folha de estilo utilizada para estilizar a interface do usuário.
JavaScript: Linguagem de programação utilizada para adicionar dinamismo à aplicação.
Estrutura do Projeto
O código está organizado da seguinte maneira:

HTML: Define a estrutura básica da página, os elementos do formulário e a área de exibição dos resultados.
CSS: Aplica estilos à página para torná-la mais atraente e amigável.
JavaScript: Gerencia a lógica de negócios da aplicação, validação de entrada do usuário e exibição dinâmica dos resultados.

Como Utilizar
Abra o arquivo index.html em um navegador da web.
Preencha o formulário com a descrição da conta e o valor a pagar.
Clique no botão "Registrar" para adicionar a conta à lista.
A lista de contas e o total serão atualizados automaticamente.

Estilos e Layout
A interface do usuário é estilizada de forma simples e responsiva, proporcionando uma experiência de usuário agradável em diferentes dispositivos. O design utiliza tons suaves e uma estrutura de layout centrada para melhorar a legibilidade e a usabilidade.

Lógica de Programação
A lógica do programa é implementada em JavaScript, onde a função registrarConta é chamada quando o formulário é enviado. Esta função valida os dados inseridos, calcula o total e atualiza dinamicamente a lista de contas na página.

Esperamos que este programa seja útil para o registro fácil e rápido de suas contas mensais! Sinta-se à vontade para explorar, personalizar e contribuir para o projeto.
